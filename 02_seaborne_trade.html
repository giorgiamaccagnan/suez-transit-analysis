Chart 2 — Seaborne Trade by Cargo Type (1990–2023)¶

Thesis Section: 1.2 — The Physical Infrastructure of Global Trade

Data Sources:

  • UNCTAD, Review of Maritime Transport (various years, 2000–2025)
  • UNCTAD Stat database: International seaborne trade by type of cargo

Description: Stacked area chart decomposing global seaborne trade (billion tons) into four cargo categories: crude oil and petroleum products, main dry bulks (iron ore, coal, grain), other dry cargo, and containerised goods. Shows the structural shift toward containerised trade.

import pandas as pd
import matplotlib.pyplot as plt
import matplotlib.ticker as mticker
import seaborn as sns

sns.set_theme(style="whitegrid", font="serif")
plt.rcParams.update({
    'font.family': 'serif',
    'font.serif': ['Times New Roman', 'DejaVu Serif'],
    'font.size': 11,
    'axes.labelsize': 12,
    'axes.titlesize': 14,
    'figure.dpi': 150,
    'savefig.dpi': 300,
    'savefig.bbox': 'tight'
})

df = pd.read_csv('../data/seaborne_trade.csv')

fig, ax = plt.subplots(figsize=(12, 6))

colors = ['#2b5c8a', '#c0392b', '#7f8c8d', '#e67e22']
labels = ['Oil & Petroleum Products', 'Main Dry Bulks', 'Other Dry Cargo', 'Containerised Goods']
cols = ['oil_tanker', 'main_bulk', 'other_dry', 'container']

ax.stackplot(df['year'], 
             [df[c] for c in cols],
             labels=labels,
             colors=colors,
             alpha=0.85)

ax.set_xlabel('Year')
ax.set_ylabel('Billion Tons', fontweight='bold')
ax.set_title('Global Seaborne Trade by Cargo Type (1990–2023)', fontweight='bold', pad=15)
ax.set_xlim(1990, 2023)
ax.set_ylim(0, 14)

# Add total line
ax.plot(df['year'], df['total_billion_tons'], color='black', linewidth=1.5, linestyle='--', alpha=0.6, label='Total')

ax.legend(loc='upper left', framealpha=0.9, fontsize=10)

# Key milestone annotations
ax.annotate('Container revolution\naccelerates', xy=(2002, 6.5), xytext=(1995, 9),
           fontsize=9, arrowprops=dict(arrowstyle='->', color='gray'),
           bbox=dict(boxstyle='round,pad=0.3', facecolor='lightyellow', edgecolor='gray', alpha=0.8))

ax.annotate('12.3B tons\n(2023)', xy=(2023, 12.3), xytext=(2019, 13.2),
           fontsize=9, ha='center', arrowprops=dict(arrowstyle='->', color='gray'),
           bbox=dict(boxstyle='round,pad=0.3', facecolor='lightyellow', edgecolor='gray', alpha=0.8))

plt.tight_layout()
plt.savefig('../figures/fig_1_2_seaborne_trade.png')
plt.savefig('../figures/fig_1_2_seaborne_trade.pdf')
plt.show()
print("✅ Chart saved to figures/fig_1_2_seaborne_trade.png and .pdf")
